How to run a project:

1. Open the terminal in your code editor and enter the following commands:

```yarn install```

2. ```yarn prepare```

3. Set environment variables.
   example:

```
FRONT_PORT=3000
ADMIN_PORT=3001
BACKEND_PORT=4000
```

4. run ```yarn dev```
